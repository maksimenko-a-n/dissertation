\begin{center}
\bf
	Многогранники двойных покрытий
\end{center}

\begin{definition}
\emph{Многогранником двойных покрытий} называется выпуклая оболочка множества
\[
  \DCP_d = \bigl\{ x\in \{0,1\}^d \mid B x = \vec{2}\bigr\},
\]
где $B \in \{0, 1\}^{m \times d}$~--- 0/1"~матрица, каждая строка которой содержит ровно четыре единицы, 
$\vec{2}$~--- $m$"~мерный вектор, все координаты которого равны $2$.
\end{definition}

Известно~\cite{Matsui:1994}, что задача проверки несмежности двух произвольных вершин многогранника двойных покрытий NP"~полна.
На самом деле, в~\cite{Matsui:1994} было описано семейство таких многогранников специального вида со следующими свойствами. 
Каждый многогранник из этого семейства обязательно содержит две диагонально противоположные вершины $d$"~мерного 0/1"~куба.
Координаты этих двух вершин заранее известны и одинаковы для всех многогранников семейства.
Указанные две вершины смежны тогда, и только тогда,
когда у данного многогранника нет других вершин.
Задача выявления вершин многогранника, отличающихся от двух уже известных,
NP-полна.
Ниже будет показано, что каждый из многогранников этого семейства,
за исключением многогранников состоящих только из двух вершин,
не является гранью ни для одного из многогранников упаковок множеств.
Для этого нам понадобится подробное описание данного семейства.

Пусть $M \in \{0, 1\}^{p \times q}$~--- 0/1"~матрица, каждая строка которой содержит ровно три единицы.
Тогда выпуклая оболочка множества
\[
  P(M) = \bigl\{ z\in \{0,1\}^q \mid M z = \vec{1}\bigr\}
\]
является многогранником разбиений множества.
(В данном случае мы заранее знаем, что каждый из $p$ элементов базового множества содержится ровно в трех из заданных по условию 
подмножеств.)
Известно, что задача проверки того, что $P(M)$ не пусто, 
NP"~полна~\cite{GareyJohnson, Karp}(???).

Для каждой $M$ построим матрицу $B$ из определения многогранника двойных покрытий следующим образом.
Положим $m = p + 2q$, $d = 3q + 5$.
Для пяти координат вектора $x \in \DCP_d$ введем особые обозначения $y_1, y_2, y_3, a, b$. 
Каждой координате $z_j$ вектора $z \in P(M)$ будут соответствовать три координаты $x_j$, $\hat{x}_j$, $x'_j$,
и два ограничения
\begin{align}
 a + b + x_j + \hat{x}_j &= 2, \label{eq:ab}\\
 y_2 + y_3 + x'_j + \hat{x}_j&= 2.
\end{align}
А для каждого ограничения вида $z_i + z_j + z_k = 1$ из описания $P(M)$
добавим к описанию многогранника $\DCP_d$ уравнение
\[
  y_1 + x_i + x'_j + x'_k = 2.
\]

Далее перейдем к рассмотрению грани этого многогранника,
полагая 
\[
a = 1, \quad b = 0.
\]
Тогда ограничение~\eqref{eq:ab} превратится в
\[
 x_j + \hat{x}_j = 1.
\]

Эта грань гарантированно содержит две вершины, 
однозначно определяемые при условии $y_2 = y_3$.

Если $y_2 = y_3 = 0$, то при всех $j$ $\hat{x}_j = x'_j = 1$, $x_j = 0$, $y_1 = 0$. Обозначим эту вершину $x^0$.

Если же $y_2 = y_3 = 1$, то при всех $j$ $\hat{x}_j = x'_j = 0$, $x_j = 1$, $y_1 = 1$. Обозначим эту вершину $\bar{x}^0$.
Очевидно, $\bar{x}^0 = \vec{1} - x^0$.

Разнообразие возникает только если 
\begin{equation}
y_2 + y_3 = 1. \label{eq:y2y3}
\end{equation}
В таком случае получаем
\( x'_j + \hat{x}_j = 1\),
а значит 
\[
 x_j = x'_j.
\]
Заметим, что для каждого $z \in P(M)$ можно построить
два вектора $x^1, x^2 \in \DCP_d$, положив 
\(y_1 = 1\) и \(x_j = z_j\) для каждого из них.
Отличие же между $x^1$ и $x^2$ возникнет при выборе значений 
$y_2$ и $y_3$ так, чтобы выполнялось условие~\eqref{eq:y2y3}.
Кроме того, наряду с векторами $x^1$ и $x^2$
наш многогранник будет содержать их дополнения
$\bar{x}^1 = \vec{1} - x^1$ и $\bar{x}^2 = \vec{1} - x^2$.

Нетрудно проверить, что других вершин у рассматриваемой грани нет.
Таким образом, множество её вершин обязательно содержит пару 
вершин $x^0$ и $\bar{x}^0$,
и, возможно, некоторое количество 
четверок вершин вида $x^1$, $\bar{x}^1$, $x^2$, $\bar{x}^2$.

Воспользуемся теперь тем, что 
\[
 x^0 + \bar{x}^0 = x^1 + \bar{x}^1 = x^2 + \bar{x}^2.
\]

Покажем, что ни один из многогранников описанного семейства,
за исключением тривиального случая, состоящего из двух вершин,
не является гранью ни для одного stable set polytope.

Доказательство для случая, когда указанная грань
содержит ровно одну четверку <<дополнительных>> вершин,
уже опубликовано.
Более сложный случай отличается лишь описанием множества единичек новой пары вершин.
В опубликованном варианте $S = U_1 \symdiff U_2 \symdiff U_3$.
В новом варианте: найдется тройка $U_i$, $U_j$, $U_k$, что $S = U_i \symdiff U_j \symdiff U_k$, соответственно.
Для удобства предполагаем, что $U_i$ это то из двух множеств единичек $x^i$, $\bar{x}^i$, которое содержит некий фиксированный (<<первый>>) элемент.
Тогда $S$ обязательно содержит тот же элемент.

То, что для такого набора единичек не будут нарушаться неравенства независимого множества, следует из того, что каждый элемент $S$ содержится в нечетном числе множеств $U_i$, $i\in[n]$. Иными словами, для любых двух элементов из $S$ найдется $U_i$ или $\bar{U_i}$, которому они оба принадлежат.

Гораздо сложнее доказать, что найдется тройка $i$, $j$, $k$ такая, что $S$ будет отличаться от каждого $U_i$.
Для $n=3$ все просто 
\[
U_1 \symdiff U_2 \symdiff U_3 \not\in \{U_1, U_2, U_3\},
\]
так как все множества различны.

Предположим теперь, что $n>3$ и нечетно.
Рассмотрим тройки вида 
\[
U_1 \symdiff U_2 \symdiff U_i, \quad 3 \le i \le n.
\]
Как было замечено выше,
\[
U_1 \symdiff U_2 \symdiff U_i \not\in \{U_1, U_2, U_i\}.
\]
Предположим, что 
\[
U_1 \symdiff U_2 \symdiff U_i = U_j.
\]
Но тогда 
\[
U_1 \symdiff U_2 \symdiff U_j = U_i.
\]
Причем
\[
U_1 \symdiff U_2 \symdiff U_k \ne U_j, \quad \forall k \ne i.
\]
Иначе $U_k = U_i$, что невозможно по условию.
По тем же соображениям,
\[
U_1 \symdiff U_2 \symdiff U_k \ne U_i, \quad \forall k \ne j.
\]
Таким образом, неподходящие тройки разбиваются на пары. 
Но всего множеств $U_i$, $3 \le i \le n$, нечетное число.
То есть обязательно найдется множество $U_i$, для которого
$U_1 \symdiff U_2 \symdiff U_i$ будет отличаться от всех остальных.

\begin{center}
\bf
Свойства симметрической разности
\end{center}

\begin{enumerate}
	\item $X \symdiff Y = X \Leftrightarrow Y = \emptyset$.
	\item $X \symdiff Y = \emptyset \Leftrightarrow Y = X$.
	\item $X \symdiff Y = Z \Leftrightarrow X \symdiff Z = Y$.
	\item $(X \symdiff Y) \symdiff Z = X \symdiff (Y \symdiff Z)$.
\end{enumerate}
